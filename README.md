# Energy Demand for Space Heating in Swiss Residential Buildings

**EPFL HUM-470 -- Economic Growth and Sustainability II (Spring 2026)**

A decomposition analysis of final energy demand for space heating in Swiss residential buildings over 2000--2024, using the IPAT framework and LMDI method.

**Group 12:** Marc Bischof, Elie Bruno, Xavier Magbi, Andrea Trugenberger

## Key Findings

- **Absolute decoupling confirmed**: GDP +53%, population +26%, floor area +36%, yet space-heating energy **-17%** since 2000
- **LMDI decomposition**: intensity/technology effect (-54,469 TJ) alone offsets population growth (+37,559 TJ) -- decoupling is structural, not just warmer winters
- **Heat pumps** now lead by floor area (29%), overtaking oil (24%)
- **Renovation rate** is the dominant lever for 2050 scenarios

## Setup

Requires [uv](https://docs.astral.sh/uv/) and Python 3.12+.

```bash
uv sync
```

## Running the Analysis

The Marimo notebook downloads all data from official APIs and generates every figure:

```bash

```
