## Implementation Phases

### Phase 1: Data Pipeline & Validation (Priority: HIGH)
**Goal**: Ensure all raw data is loaded, cleaned, and validated in the Marimo notebook.

1. **Audit `notebooks/analysis.py`** — read the current notebook, identify what's done vs missing
2. **Validate BFE energy balance** — extract residential heating by carrier (oil, gas, HP, wood, district, electric) for 2000-2023
3. **Add missing data series**:
   - Dwelling count & average floor area (BFS GWR) — may need additional download
   - Heating system shares by building type (GWR energy source data)
   - GDP series (SECO/BFS) for decoupling assessment
4. **Weather correction** — compute HDD-normalized energy using Eurostat HDD data
5. **Build IPAT time-series table**: P, D/P, A/D, S_s, e_s for each year 2000-2023
6. **Export clean CSVs** to `data/processed/` for reproducibility

### Phase 2: LMDI Decomposition (Priority: HIGH)
**Goal**: Implement additive LMDI decomposition in the notebook.

1. **Implement LMDI-I additive** decomposition (Ang 2004 method)
2. **Decompose change in E_heat** into contributions from:
   - Population effect
   - Dwelling density effect
   - Floor area effect
   - Heating mix effect (fuel switching)
   - Energy intensity effect (insulation/efficiency)
   - (Optional) HDD effect
3. **Generate key charts**:
   - Waterfall chart: factor contributions 2000→2023
   - Time-series: indexed evolution of each factor (base year = 100)
   - Stacked area: heating mix evolution
   - Decoupling plot: E_heat vs GDP indexed
4. **Export figures** as SVG to `figures/` for Typst inclusion

### Phase 3: Write Section 4 — Data & Trends (Elie)
**Goal**: Present the full I=P×A×T time series with interpretation.

1. **4.1 Evolution of Key Indicators** — time-series plots with commentary
2. **4.2 Quantitative Decomposition** — LMDI results table + waterfall chart
3. **4.3 Decoupling Assessment** — absolute vs relative decoupling verdict
4. **4.4 Discussion** — interpret which factors dominated, structural breaks (2008 MuKEn, 2021 gas crisis)

### Phase 4: Write Section 5 — Policy & Efficiency Drivers (Elie)
